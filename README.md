Este es tu espacio de trabajo para aprender desarrollo frontend. El repositorio contiene:

- **Carpetas de clases** (`clase1-html-css`, `clase2-javascript`, etc.): Material de práctica personal (NO subir cambios)

## 🚀 Cómo usar este repositorio

### Configuración inicial (solo una vez)

1. **Clona este repositorio:**
   ```bash
   git clone https://github.com/heldigard/curso-frontend-fumc.git
   cd curso-frontend-fumc
   ```

2. **Abre la carpeta en VS Code:**
   ```bash
   code .
   ```

### 📖 Material de práctica (carpetas clase1, clase2, clase3...)

Estas carpetas contienen ejercicios y ejemplos para que **practiques por tu cuenta**:
- Puedes modificarlos libremente en tu computadora
- Úsalos para aprender y experimentar
- **NO subas cambios de estas carpetas a GitHub**

### ⭐ Taller (carpeta Taller/)

Esta es tu **carpeta de trabajo oficial**:

1. **Encuentra tu carpeta** en `Taller/tu_nombre_completo/`
2. **Completa las actividades** dentro de tu carpeta
3. **Sube SOLO los cambios de tu carpeta** a GitHub

### 📤 Subir tus actividades del Taller a GitHub

```bash
# 1. Verifica que solo modificaste TU carpeta
git status

# 2. Agrega SOLO tu carpeta del Taller
git add Taller/tu_nombre_completo/

# 3. Guarda los cambios con un mensaje descriptivo
git commit -m "Completadas actividades del taller"

# 4. Sube a GitHub
git push origin main
```

> ⚠️ **IMPORTANTE**: Solo debes hacer `git add` de tu carpeta dentro de `Taller/`. No subas cambios de las carpetas clase1, clase2, etc.

## 🆘 Solución de problemas

### La página no se ve bien
- ✅ Verifica que guardaste el archivo (Ctrl + S)
- ✅ Recarga el navegador (F5 o Ctrl + F5)
- ✅ Revisa la consola (F12 → Console) por errores

### El JavaScript no funciona
- ✅ Abre la consola del navegador (F12)
- ✅ Busca mensajes de error en rojo
- ✅ Verifica que completaste todas las secciones TODO

### Error al hacer git push
- ✅ Asegúrate de solo agregar tu carpeta: `git add Taller/tu_nombre/`
- ✅ No hagas `git add .` (esto agrega todo)
- ✅ Verifica con `git status` qué archivos vas a subir
